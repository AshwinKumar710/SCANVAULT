📦 NFC & QR Code Package Tracker
Objective
A streamlined solution for storing, organizing, and tracking package information with integrated QR code and NFC technologies. The app improves warehouse efficiency and package traceability.

🚀 Key Features
User Dashboard: Recent packages, delivery statuses, quick filters.

Package Profiles: Recipient, product details, address, quantity, custom attributes, photo attachments.

QR Code Generation: Auto‑generate, display, and save QR codes for each package.

NFC Tag Writing: Encode package data (JSON format) onto NFC tags with one tap.

NFC Tag Reading: Scan tags to instantly retrieve and update package details.

Tracking & Status Updates: Mark packages as Pending, In Transit, Delivered via QR/NFC scans.

Admin Settings: Multi‑user access, permissions, and activity logs.

🛠️ Tech Stack
Frontend: Flutter (cross‑platform UI)

Backend: Python

Database: Firebase Firestore

Local Storage: sqflite (for offline package data)

📦 Core Functions
CRUD operations (Create, Read, Update, Delete)

Scan QR codes

Generate QR codes

Read NFC tags

Write NFC tags
